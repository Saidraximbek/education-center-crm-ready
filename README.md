# O'quv Markazi CRM

Direktor va Admin rollari uchun yaratilgan zamonaviy React + Vite CRM. Tizim Firebase Authentication, Firestore CRUD servislar, dashboard analitika, to'lovlar, davomat, talabalar, guruhlar, fanlar, hisobotlar va qorong'i/yorug' rejimni qo'llab-quvvatlaydi.

## Ishga tushirish

```bash
npm install
npm run dev
```

`.env.example` faylini `.env` nomi bilan nusxalang va real ishlatish uchun Firebase qiymatlarini kiriting.

## Firebase sozlanmagan holatda demo kirish

- Direktor: `director@educrm.local` / istalgan parol
- Admin: `admin@educrm.local` / istalgan parol

Firebase muhit qiymatlari kiritilganda autentifikatsiya va Firestore servislar Firebase orqali ishlaydi.
